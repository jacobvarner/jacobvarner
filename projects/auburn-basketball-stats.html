Auburn Basketball Stats was my first big project using Meteor to really help me grasp the best practices for the framework. Auburn Basketball Stats aims to track all types of statistics and records for Auburn basketball and present them in a visually pleasing and easy to access way. Auburn Basketball Stats uses several different input forms that allow me to add stats from past seasons and future games. This data is stored in Mongo collections as MongoDB is the database used with Meteor at the time of this project. Data and stats are then pulled from the collections using methods called from the client that can access the server to focus on security. Meteorâ€™s live update is used throughout the site to allow for different sorting of stats. For example, on the player stats pages, users can see stats from the entire season or the most recent 3, 5, or 10 games. I will continue to add features to Auburn Basketball Stats, but I am confident in the job that I did to utilize best practices in Meteor.
